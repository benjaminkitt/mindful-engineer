  def retry(fn, attempts=5, base=0.25):
      for index in range(attempts):
          try:
              return fn()
          except Retryable:
              if index == attempts - 1:
                  raise
              sleep(base * (2 ** index) + random() * base)

This is the version I actually ship. Backoff, jitter, and one place to stop lying to yourself about how many retries you need.
