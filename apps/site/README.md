# Mindful Engineer public site

This is the separate Astro app for the public reading surface.

## Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment

This app is configured for static output and is intended for Cloudflare Pages.

- Build command: `npm run build`
- Output directory: `dist`
- Root directory: `apps/site`

The site URL is `https://mindful.engineer`.
